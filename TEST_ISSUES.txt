【BUG-001】严重 | 容器初始化导致测试套件挂起 (Test Suite Hang)
--------------------------------------------------------------------------------
级别: P0 - 阻塞性
位置: src/wechat_summarizer/infrastructure/config/container.py:80
现象: 当测试用例触发 Container.summarizers 属性时，_create_summarizers() 方法
      在执行过程中长时间阻塞（>4分钟），导致 pytest 测试套件挂起，最终需要
      KeyboardInterrupt 才能终止。
影响范围: 影响所有依赖 Container 的测试：test_graphrag.py, test_summary_evaluator.py
         (integration部分), test_container.py, integration/test_full_flow.py
根因分析: 容器使用双重检查锁（Double-Check Locking）懒加载模式，_create_summarizers()
         内部可能尝试连接外部服务（如 Ollama localhost:11434）或加载大型模型/依赖
         （如 sentence-transformers），在无服务可用时超时等待。
建议修复:
  1. 为 _create_summarizers() 中的外部连接添加快速失败超时（如 3 秒）
  2. 在测试 conftest.py 中提供 mock Container，避免触发真实服务连接
  3. 考虑添加 Container.create_minimal() 工厂方法用于测试环境
参考标准: OWASP Testing Guide - 测试隔离原则

【BUG-002】中等 | 已弃用 API 使用: datetime.utcnow()
--------------------------------------------------------------------------------
级别: P2 - 重要
位置: src/wechat_summarizer/infrastructure/adapters/exporters/onenote.py:464
现象: 使用了 Python 3.12+ 中已弃用的 datetime.datetime.utcnow()
代码: created = escape(datetime.utcnow().isoformat())
影响: Python 3.14 运行时产生 DeprecationWarning；未来 Python 版本将移除此 API，
     导致运行时错误。
建议修复: 替换为 datetime.datetime.now(datetime.UTC).isoformat()
参考标准: Python 3.12 Release Notes - Deprecated APIs

【BUG-003】中等 | Mypy 类型错误: progress.py 缺少 Optional 导入
--------------------------------------------------------------------------------
级别: P2 - 重要
位置: src/wechat_summarizer/shared/progress.py:269, :302, :325
现象: 使用了 Optional 类型标注但未导入 typing.Optional，导致运行时 NameError。

【WARN-002】安全 | 加密密钥使用固定盐值
--------------------------------------------------------------------------------
级别: P2 - 安全关注
位置: src/wechat_summarizer/shared/utils/security.py:59
现象: PBKDF2 密钥派生使用固定盐值 b"wechat_summarizer_v2"
影响: 同一用户在同一机器上始终生成相同密钥，降低了密钥的抗彩虹表能力。
建议: 使用 os.urandom(16) 生成随机盐值并持久化保存
参考标准: OWASP Cryptographic Storage Cheat Sheet

【WARN-005】安全 | SSRF 防护不检查 DNS 重绑定
--------------------------------------------------------------------------------
级别: P3 - 安全改善
位置: src/wechat_summarizer/domain/value_objects/url.py:81-83
现象: _is_private_address() 方法注释说明不进行 DNS 解析以避免网络请求。
     这意味着攻击者可以通过 DNS 重绑定（DNS Rebinding）绕过 SSRF 防护：
     使用一个域名首次解析为公网 IP，后续解析为内网 IP。
建议: 在实际发起 HTTP 请求前（而非 URL 构造时）进行连接级 IP 检查
参考标准: OWASP SSRF Prevention Cheat Sheet
